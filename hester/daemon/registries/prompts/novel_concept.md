2. **Ground in context** - How might this apply to this specific codebase?

## Research Guidelines
- Start with authoritative sources (official docs, reputable tech blogs)
- Look for real-world usage examples and case studies
- Note version requirements and compatibility concerns
- Identify common pitfalls and gotchas

## Connecting to Codebase

- Identify where the concept would integrate
- Note what existing code might need to change
- Consider how it fits with current architecture
